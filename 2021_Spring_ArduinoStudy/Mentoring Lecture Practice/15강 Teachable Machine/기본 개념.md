## **Teachable Machine**
: 가르쳐 줄 수 있는 인공지능. (Machine==인공지능. 인공지능을 학습할 때 사용하는 방법론 중 하나가 Machine Learning이기 때문)
1. image project
2. audio project
3. pose project

## **Epochs(에포크)란?**
- 1 epoch: 전체 데이터 셋에 대해 한 번의 학습 과정이 완료되었다는 것(epochs=10 이면 전체 데이터를 10번 사용해서 학습을 거치는 것)
- underfitting: epoch 적음
- overfitting: epoch 많음
